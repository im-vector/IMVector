### imvector

IMVector I'm Vector 向量协议通信

### 特性

1. 简单性和易用性
    - 能快速部署使用
2. 可定制性
    - 模块可重写
3. 可扩展性
    - 模块可安装
4. 稳定性可靠性
    - 按预期的状态持续运行
5. 支持分布式部署
